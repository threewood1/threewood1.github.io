Git Bash Here 不见啦！！

发表于2023-09-22|更新于2023-09-22|Fix
|阅读量:

不见啦！！

其实已经遇到好几次了，每次电脑更新，系统都自动把我鼠标右键的 Git Bash Here 弄没了（应该是Windows Defender被当作威胁给清理掉了）。所以就有了这个解决方法。可以只看图片！

我的解决方法

step1

打开注册表：键盘输入：win + R，然后输入regedit后回车。

step2

设置git-bash.exe路径。找到这个路径计算机\HKEY_CLASSES_ROOT\Directory\Background\shell (这里注意下面还有shell，不要选错了)，右键shell并选择新建-项命名为Git Bash Here，然后点击Git Bash Here并右键新建-项命名为command，结构如下图所示：

然后修改command中的值为自己的git-bash.exe的所在目录，我的地址是这样：

step3

最后一步设置图标。点击Git Bash Here，右键新建字符串值，取名Icon,目的是为了给Git Bash Here加一个图标好看一些。修改值为安装路径D:\Program Files\Git加上\mingw64\share\git\git-for-windows.ico。这里找到自己的路径加上去即可。

成功修复！

这样就完成了，希望这个方法对你有帮助。
